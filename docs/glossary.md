## Glossary

### Terms, Expressions e Acronyms (TEA) organized alphabetically.

| TEA (PT)                 | Definition                                                                                                                              |
|--------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| **Address**              | Set of information that indicates a location: Street name, Door number, Zip code, City.                                                 |
| **Analysis**             | Recruitment process phase where the customer manager uses all the collected data to rank candidates.                                    |
| **Application**          | Candidate intention of applying to a job offer, by the means of documents and/or letters.                                               |
| **Candidate**            | Person that has the intention of being placed on a given job position.                                                                  |
| **Clients**              | Same as customer.                                                                                                                       |
| **Contract Type**        | Formal document with clauses that set the benefits and duties of each signing part, to each other.                                      |
| **Customer**             | Person that has the intention of hiring a given candidate, by placing job openings.                                                     |
| **Customer Manager**     | The job role of the person who manages the customer(s) and is also responsible for the interviews with the candidates.                  |
| **Entity**               | Same as customer. It can be private or public, a single person or a group, such as a company.                                           |
| **Function**             | A name that points to job role main function. Same as Title.                                                                            |
| **Human Resources**      | Resource in the form of a human with a specific skill set.                                                                              |
| **Interviews**           | Formal moment with customer manager and/or with customer, to explore more details about the candidate.                                  |
| **Jobs4U**               | Name of the company. Company specialized in talent acquisition.                                                                         |
| **Job Opening**          | A formal intention of publicising a job vacancy, by the means of job requirements, and through one or more communication channels.      |
| **Job Position**         | Same as job opening.                                                                                                                    |
| **Job Reference**        | The id associated to a job opening.                                                                                                     |
| **Mode**                 | Type of work setting, such as remote, hybrid, or onsite.                                                                                |
| **Operator**             | The job role of the person who validates candidates applications and is responsible to establish contact with them, whenever necessary. |
| **Recruitment Process**  | Set of phases that comprise the recruitment. It can be from the application phase up until the proposal or even the placement.          |
| **Recruitment Services** | A service that provides talent selection and/or attraction based on certain criteria set by a job opening.                              |
| **Result**               | Analysis result, comprised by the application result and by the interview result, whenever applicable.                                  |
| **Screening**            | Second phase of the recruitment process. All the applications are verified against a set of requirements.                               |
| **System Admin**         | The job role of the person who administrates one or more digital systems.                                                               |
| **Talent Acquisition**   | Finding persons with characteristics that match the requirements for a given job role.                                                  |
| **Title**                | A name that points to job role main function. Same as Function.                                                                         |
